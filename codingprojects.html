About Replit

Replit is an online IDE that makes cloning these projects from my GitHub easy and intuitive. Replit does this by running the all code on a virtual machine. As a result, the code is a bit slow and their embedding interface does lack some customization. If you're interested in a speedier version I encourage you to clone them directly.

From my own testing the plotting through Replit only seems to work on recent versions of Google Chrome (not Incognito mode) and Firefox.

Rearrangements

The Riemann series theorem (or Riemann rearrangement theorem) loosely states that for any conditionally convergent series, and any [extended] real number, there exists a rearrangement of the series that converges to the given real number. The proof of this beautiful theorem given in Rudin's book on analysis actually gives an algorithm for constructing the desired rearrangement. Here, I implement this algorithm and plot the progression of the partial sums of the rearranged series using a few options for conditionally convergent series.

To run the code make sure you're on an compatible browser, and that the Console tab is active. Press play, type your inputs into the console, and follow the instructions given in the console. Note that for real numbers larger (more negative than) 8 (-8) the simulation can take a very long time to converge.
